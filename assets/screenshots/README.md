# CCW Map Screenshots

This directory contains screenshots for the Google Play Store listing and promotional materials.

## Play Store Screenshots

These screenshots are used in the Play Console under "Main store listing > Phone screenshots".

### Screenshot Order and Descriptions:

1. **01-sign-in.png** (1.1 MB)
   - Sign In / Create Account screen
   - Shows app branding, map icon, and tagline
   - Clean authentication UI with email/password fields

2. **02-map-view-with-pins.png** (467 KB)
   - Main map interface with multiple location pins
   - Shows green pins (allowed zones) and red pins (restricted zones)
   - Displays real-world location context (Tampa, FL area)
   - Demonstrates core mapping functionality

3. **03-create-pin-dialog.png** (433 KB)
   - "Create Pin" dialog interface
   - Shows location name input field
   - Color-coded carry zone status options (Allowed, Uncertain, No Guns)
   - Optional security details checkboxes
   - Clean, user-friendly form design

4. **04-edit-pin-allowed.png** (391 KB)
   - "Edit Pin" dialog for an allowed location (Publix)
   - Shows green "Allowed" status selected
   - Demonstrates edit and delete functionality
   - Save/Cancel action buttons

5. **05-edit-pin-restricted.png** (90 KB)
   - "Edit Pin" dialog for a restricted location (Saint Joseph's Hospital-North)
   - Shows red "No Guns" status selected
   - Displays restriction reason dropdown (Healthcare Facility)
   - Shows optional detail checkboxes (Active security screening, Posted signage visible)
   - Demonstrates detailed restriction tagging
