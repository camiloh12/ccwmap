## Conventions

- Every external-facing link should carry a `?utm_source=...` parameter so you can attribute installs.
- Templates use `{{double-curly placeholders}}`. Find-and-replace before sending.
- Don't paste the same text into multiple Reddit subs — community/algorithm both detect duplicates.
